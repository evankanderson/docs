Knative for Platform Administrators

Knative extends Kubernetes, seamlessly.

Knative integrates with your existing Kubernetes tools:

  • Serving supports pluggable HTTP ingress, including service mesh. Works with your existing infrastructure investments.
  • Eventing support pluggable event transport, providing configuration-driven routing for your existing infrastructure.
  • Standard resource model works with Kubernetes-native policy tools like quotas and OPA Gatekeeper.
  • Integration with Cert Manager for automatic SSL provisioning.

Knative is Kubernetes done right:

  • Tested High Availability with sharding and leader election. We test our active-active controller configuration by terminating elected leaders every 30s while under load.
  • Configuration, simplified. Sensible, configurable defaults make it easy to write configuration without cutting off the power that you need.
  • Improve efficiency. Multi-tenant and autoscaling components as well as scale-to-zero in Serving use cluster resources more efficiently.
